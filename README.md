Landing page estática para Vortex Hub.

Esta versión está diseñada para ser publicada en Netlify sin backend adicional, con un CTA directo al formulario de Brevo.

Pasos rápidos:
1. Reemplaza el enlace `https://forms.brevo.com/tu-formulario` con el URL real de tu formulario Brevo.
2. Sube el proyecto a un repositorio Git.
3. Conecta el repositorio a Netlify y despliega como sitio estático.
